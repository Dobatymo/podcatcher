# Podcatcher

Simple podcast / RSS download app.

## Requirements
- Python 3.7 (or newer)

## Install

- `pip install .`

## Run

- GUI: Run `podcatcher-web` (or `python -m podcatcher.web`) and open `localhost:8000` in your browser to connect to the GUI.
- CLI: Run `podcatcher-cli` (or `python -m podcatcher.cli`).
